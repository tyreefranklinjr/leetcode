# Daily Challenges

## Overview

This folder contains my daily LeetCode problem-solving practice.

Each problem includes:
- A clear problem statement
- Examples and constraints
- Clean formatting for readability
- (Optionally) optimized solutions and notes

The goal of this collection is to:
- Strengthen problem-solving skills
- Improve algorithmic thinking
- Prepare for technical interviews
- Build consistency through daily practice

---

## Structure

Each file represents a single problem and is formatted in Markdown for GitHub readability.

Typical structure inside each file:
- Problem description
- Examples
- Constraints
- (Optional) Solution approach and code

## Topics Covered

Problems span across common data structures and algorithms, including:

- Hash Maps & Sets
- Two Pointers
- Stacks & Queues
- Recursion
- Sorting & Searching
